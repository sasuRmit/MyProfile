My Project

My project aims to be an interactive and engaging fitness tracking application on IOS devices by including a number of puzzle games and rewarding users for updating a number of daily fitness trackers with rewards for the games.

One of the most difficult things for a new application is to retain active users. By rewarding users for using the fitness and wellbeing trackers users are more likely to be active and continue to use the application. This will also help people improve there quality of life.

The application will have two main functions, tracking user fitness activity and wellbeing, and puzzle games to keep the user in the application. It will track a number of activity stats available from the IOS Health app which includes steps taken, distance travelled and flights of stairs climbed, this data will be available to the user in a number of different reports to give them insights into their activity. The user will also be able to add and track exercise, fluid and food consumption, and weight. The puzzle games will include a number of different puzzles at a varying degree of difficulty so the user can play a relaxing game or really challenge themselves. The games will also have trackers that will allow the user to see if they are able to compete puzzles quicker and in a smaller amount of moves and can compare their scores with other users if desired.

When the user updates their exercise and food trackers they receive small bonuses the assist in certain puzzles. This is to encourage the user to be more active with tracking items. The encouragement will hopefully mean that the user continues to track items they will be able to use the data and reports to see any patterns and make any changes if necessary.
